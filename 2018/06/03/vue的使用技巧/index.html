  • 问题三：src/assets 和 static ，存放静态文件有何区别？

assets中静态文件：assets下的静态文件会视作模块依赖，可以使用相对路径（同样也可以使用绝对路径，只不过不做处理），webpack会处理好路径问题；这个不仅仅可以放在assets下，只要放在src目录下即可；文件引入方式：相对路径，绝对路径，require(‘./relative/path/to/file.jpg’)

static中静态文件：Webpack根本不处理文件：它们直接被复制到最终目的地，并且具有相同的文件名（打包后不做改变）;必须使用绝对路径来引用这些文件;

vue之间的通信

  • 父级向子级通讯(props)
    parent: data.ParentMsg

    调用： <v-child :childProps='ParentMsg'></v-child>

    child: props[childProps]
    组成：属性(child.props),子组件调用(v-child),属性绑定（v-bind:child.props = ParentMsg)

  • 子级向父级通讯
    parent: methods.changeParentData();

    parent调用： <v-child v-on:childEmit = 'changeParentData'>
child组件: <template><div><botton @click="childFired"></botton></div></template>

    child: methods.childFired(() => { this.$emit(‘childEmit’,childArguments)})
    组成：事件触发器,并触发emit(child-event)，监听器(parent-listener)，执行方法（parent-function）
